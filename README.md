# Battle of Glory

Un juego de arcade espacial donde 4 Cadetes de diferentes planetas luchan por la gloria suprema.

## Descripción

Battle of Glory es un juego de arcade inspirado en los mejores títulos de casinos modernos. Combina acción, estrategia y gráficos vectoriales en un formato ligero y rápido de cargar.

## Cómo jugar

- Abre el archivo `index.html` en tu navegador web
- O visita: https://pilukarts.github.io/BattleofGlory/

## Los 4 Cadetes

1. **Inferno** - El Cadete de fuego con ataques explosivos
2. **Glacier** - El Cadete de hielo con poderes congelantes
3. **Viper** - El Cadete venenoso con ataques rápidos
4. **Celestial** - El Cadete cósmico con habilidades especiales

## Características

- Gráficos 100% en SVG (escalables y ligeros)
- 4 personajes jugables con estilos únicos
- Sistema de bonificaciones y combos
- Diseño responsivo para móvil y escritorio
- Código abierto y extensible

## Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Animaciones)
- JavaScript ES6
- SVG para gráficos
